## Multimodal-RAG

## How to run application?

## Creating Environment 
```bash
conda create -p venv python=3.10 -y

```
## Activating Environment
```bash
source activate venv/
```

## Installing dependencies
```bash
pip install -r requirements.txt
```
### To run the application
```bash
python app.py 
```
